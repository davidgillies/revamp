{% trans "Introduction" %}

{% trans "This tool enables the estimation of the resource recovery potential of organic waste streams in a city/town. The current version of the tool includes three waste streams and four resource recovery options. To use the tool, follow the step-wise instructions described below." %}

{% trans "Step One" %}

{% trans "The WasteQuality tab has default data on the physical and chemical characteristic of the waste streams. If you have data available for some or all of the various parameters in your city/town, please fill that data in and replace the default values. Otherwise, you can maintain the default values." %}

{% trans "Step Two" %}

{% trans "The TreatmentProcesses tab has default data on the physical and chemical transformation of waste streams as they go through the treatment processes. If you have data available for some or all of the various parameters in your city/town, based on previous experience with the resource recovery technologies, please fill that data in and replace the default values. Otherwise, you can maintain the default values." %}

{% trans "Step Three" %}

{% blocktrans %} The Prices tab has default data on the potential prices that resource recovery products can be sold at on the local market. If some or all of these products are already being sold in your city/town, please use the applicable price values. Otherwise, you can estimate potential prices in dialogue with stakeolders, basing on the prices of similar or equivalent products that could be substituted or complimented by the resource recovery products.{% endblocktrans %}

{% trans "Step Four" %}

{% trans "Enter the amounts of waste streams that you have available in your city/town and determine what percentage of each waste stream should be used for each resource recovery option." %}

{% trans "Step Five" %}

{% trans "When ready press the Calculate Results button below. This will generate the results which will appear below the Inputs box. The Results Tab will then be populated with results indicating the resource recovery potential for each option and data charts will appear on the Charts tab. The Graphs tab contains bar graphs that you can use to compare the various resource recovery options on the basis of their revenue potential, the amounts of nutrients that can be recovered and the energy amount that can be recovered." %}

{% trans "Waste Streams" %}

{% trans "Faecal Sludge" %}

{% trans "Faecal sludge comes from onsite sanitation technologies, i.e., it has not been transported through a sewer. It results from the collection and storage/treatment of excreta or blackwater, with or without greywater. Faecal sludge includes both sludge from pit latrines and that from septic tanks." %}

{% trans "Sewage Sludge" %}

{% trans "Sewage sludge (also referred to as wastewater sludge) is sludge that originates from sewer-based wastewater collection and (semi-) centralized treatment processes." %}

{% trans "Organic Municipal Solid Waste" %}

{% trans "This is the organic part of the urban solid waste and it includes items like food waste, market waste and crop residues" %}

{% trans "Resource Recovery Options" %}

{% trans "Anaerobic Digestion" %}

{% trans "This is a process which generates biogas, which has about 60% methane content. Biogas can be used for lighting, cooking and also for generating electricity and heat. The process of anaerobic digestion also generates a residue which can be used as soil conditioner or fertilizer in a farm to recover nutrients like nitrogen, phosphorus and potassium." %}

{% trans "Solid Fuel" %}

{% trans "Excreta and organic waste streams have a high calorific value and can be turned into a solid dry fuel for combustion in briquette or powder form. This can be used either for cooking in households and institutional kitchens as well as for industrial applications like kilns and boilers." %}

{% trans "Black soldier fly larvae" %}

{% trans "Organic waste streams can can be treated using fly larvae composting, for example with the Black Soldier Fly, to produce valuable larvae and a residue. The larvae of the black soldier fly is 40% protein and 30% fat and can therefore make a protein-rich animal feed. The residue from the fly larvae composting contains nutrients and can be applied to a garden as soil conditioner or fertilizer." %}

{% trans "Composting" %}

{% trans "This is a process whereby a waste stream is composted to make soil conditioner or fertilizer for applying on farms. The compost generated is often rich in nutrients and also rich in organic matter content." %}
